### **Task: Build and Run a Docker Container**

In this task, your goal is to build and run a Docker container using the provided `Dockerfile` and `main.py` files.

### **Instructions:**

1. **Download or Place the Files**: Ensure that you have the following files in your working directory:
   - `Dockerfile`
   - `main.py`

2. **Build the Docker Image**: Using the provided `Dockerfile`, build a Docker image for the application.

3. **Run the Docker Container**: After building the image, run a container from that image.

4. **Verify the Container**: Check if the container is running properly and access any output if necessary.
